Technology Stack

The most important two are Angular and Firebase. I have experience with AngularJS previously, so Angular is an easy choice. As of Firebase, it is an awesome platform for web/mobile application (and its free quota is very generous). The rest of the stack is chosen around these two.

  1. Angular: primary web framework.
  2. Angular Material: UI framework.
  3. AngularFire2: Angular library for Firebase.
  4. flex-layout: UI layout for Angular.
  5. Jasmine, Karma and Protractor: Unit tests and End to End (e2e) tests.
  6. Firebase: Firestore as the main database, Hosting, Function and Storage.
  7. Typedoc: Typescript document generator.
  8. Docusaurus: Design documents (this document).
  9. CircleCI: CI/CD platform.
